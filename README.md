# Pokemon-battle-simulator
C++ Pokémon battle simulator using JSON data, currently hardcoded teams, compares stats to determine winners
A C++ project where I simulate Pokémon team battles using real stat data from a JSON Pokédex.

## Features
- Reads Pokémon data from JSON
- Simulates battles between two teams
- Compares total stats to determine winner

## CONS
-It does not checks for the Pokemon's type
-It only compares the pokemon's BST
